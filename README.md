# Rails 權限系統

- ruby 3.0.0
- bundler 2.2.3
- Rails 6.1.4.1
- DB: postgresql

gems
```ruby
gem 'devise', '~> 4.8'
gem 'pundit', '~> 2.1', '>= 2.1.1'
```

start
```code
  bundle install
  yarn install
  rails db:setup
  rails s
```

See db/seed.rb for init datas.
